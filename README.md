# Multitask
A discord bot that connects to the guilded api and can do malicous things.

## Setup
```
{
  "discord": {
           "token": "" // Discord Token
  }, "guilded": {
           "email"   : "", // Guilded Email
           "password": "", // Guilded Password
           "referrer": ""  // [Optional] Guilded Referrer
  }
}
```
